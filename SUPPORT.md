# Support

## Before opening an issue

1. Read the [README](README.md) and the [website documentation](https://sanjit-k.github.io/ftc-toolchain/docs/).
2. Update to the current package release.
3. Run `npx ftc-toolchain doctor [projectPath]` and remove any credentials or personal information before sharing its output.
4. Search [existing issues](https://github.com/Sanjit-K/ftc-toolchain/issues).

Use the **Question or setup help** issue form when the documentation does not resolve the problem. Include your operating system, AI client, Node version, FTC SDK project layout, and the exact error. A minimal example is much easier to diagnose than an entire team repository.

Support is provided by volunteers on a best-effort basis. Response times are not guaranteed, especially during competition season.

## Where to report different problems

- Reproducible defect: **Bug report** issue form.
- New capability or workflow: **Feature request** issue form.
- Incorrect or unclear docs: **Documentation issue** form.
- Suspected vulnerability: [private security advisory](https://github.com/Sanjit-K/ftc-toolchain/security/advisories/new).
- Conduct or student-safety concern: follow [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

FTC Toolchain cannot provide emergency field support and is not affiliated with FIRST or REV Robotics. Follow the current competition manual and official hardware guidance.
